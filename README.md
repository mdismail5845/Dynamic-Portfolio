# Dynamic Portfolio

A fully responsive and dynamic portfolio web application built with Django. This project allows users to showcase projects, categories, and portfolio sections dynamically from the admin panel. Designed for developers and freelancers to manage and display their work efficiently.

![Django Version](https://img.shields.io/badge/django-5.0+-green.svg)
![Python Version](https://img.shields.io/badge/python-3.10+-blue.svg)
![License](https://img.shields.io/badge/license-MIT-red.svg)

## Features

- **Dynamic Content Management** – Add, update, or remove projects, categories, and portfolio sections via Django’s built-in admin panel
- **Fully Responsive Design** – Optimized for desktop, tablet, and mobile devices
- **Project Showcase** – Display projects with titles, descriptions, images, and links
- **Category Filtering** – Organize projects into categories for better navigation
- **Clean & Modern UI** – Professionally designed layout to highlight your work
- **Easy to Customize** – Built with Django’s modular structure for simple modifications
